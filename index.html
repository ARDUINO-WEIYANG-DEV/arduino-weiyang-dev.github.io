温湿度检测仪

Using an Arduino Uno with a DHT11 and an OLED display to build a temperature‑and‑humidity monitor is not complicated. The core idea is simple: let the Arduino read data from the DHT11 sensor and send it to the OLED screen via the I²C protocol for display.

1. Hardware Connections

  • DHT11 temperature & humidity sensor:
    • VCC → 5V (Arduino)
    • GND → GND (Arduino)
    • DATA → Digital pin 2 (Arduino)
  • 0.96″ OLED display (I²C interface):
    • VCC → 5V (Arduino)
    • GND → GND (Arduino)
    • SCL → A5 (Arduino)
    • SDA → A4 (Arduino)

2. Software Preparation (Installing Libraries)

  1. Open the Arduino IDE, go to Sketch → Include Library → Manage Libraries….
  2. Search for and install the following libraries:
    • Adafruit GFX Library
    • Adafruit SSD1306
    • DHT sensor library (usually provided by Adafruit)

3. Example Code

Copy the code below into your IDE, make sure the board is set to Arduino Uno, select the correct port, and click Upload.

#include <Wire.h>
#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>
#include <DHT.h>

// ------- Configuration parameters --------
#define DHTPIN 2          // Digital pin connected to DHT11 data
#define DHTTYPE DHT11     // Sensor type

#define SCREEN_WIDTH 128  // OLED width
#define SCREEN_HEIGHT 64  // OLED height
#define OLED_RESET -1     // Reset pin (not used for I2C, set to -1)
#define OLED_ADDR 0x3C    // I2C address (most common is 0x3C)
// ---------------------------------------

DHT dht(DHTPIN, DHTTYPE);
Adafruit_SSD1306 display(SCREEN_WIDTH, SCREEN_HEIGHT, &Wire, OLED_RESET);

void setup() {
  Serial.begin(9600);
  
  // Initialize DHT11
  dht.begin();

  // Initialize OLED
  if(!display.begin(SSD1306_SWITCHCAPVCC, OLED_ADDR)) {
    Serial.println(F("SSD1306 initialization failed!"));
    for(;;); // Halt execution
  }
  
  display.clearDisplay();
  display.setTextSize(1);          // Set font size
  display.setTextColor(SSD1306_WHITE);
  display.display();
}

void loop() {
  delay(2000); // Wait 2 seconds for sensor stabilisation

  float h = dht.readHumidity();    // Read humidity
  float t = dht.readTemperature(); // Read temperature (Celsius)

  // Check if readings are valid
  if (isnan(h) || isnan(t)) {
    Serial.println(F("Sensor read failed!"));
    return;
  }

  // ----- Display data on OLED -----
  display.clearDisplay();

  display.setCursor(0, 0);
  display.print("Temperature: ");
  display.print(t);
  display.println(" C");

  display.setCursor(0, 20);
  display.print("Humidity: ");
  display.print(h);
  display.println(" %");

  display.display(); // Update the screen
  // --------------------------------
}

4. Notes and Further Enhancements

  • Pull‑up resistor: Some DHT11 modules have it built in; if you buy a bare sensor, add a 10 kΩ resistor between VCC and DATA.
  • Read interval: It is recommended to wait at least 2 seconds between readings – querying too frequently may cause errors.
  • Power and enclosure: You can power the device via USB or a 9 V battery, and 3D‑print a case for a neat appearance.
  • Accuracy and expansion: For higher precision, consider upgrading to a DHT22. You can also add a Wi‑Fi module for remote monitoring, or attach a buzzer/LED for over‑temperature/humidity alerts.

Happy building! 😊
